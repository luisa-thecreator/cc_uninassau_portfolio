Projeto 1 - Wireframe do site CNN :

Para este projeto utilizamos o site da CNN para criar um wireframe no Figma, começamos estudando sua estrutura, focando na barra de navegação, manchetes e seções de notícias. No Figma, criamos uma página web padrão e adicionamos elementos básicos como a navegação, áreas de manchetes, seções de notícias e rodapé. Usando formas simples para representar imagens e textos. Depois, revisamos com stakeholders para ajustes.
